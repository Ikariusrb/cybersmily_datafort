Netrunner Cyberdeck Manager

This application is for Cyberpunk 2020 to help manage a Netrunner's deck and the programs they use. This relies upon the source book Bartmoss' Brainware Blowout. Some of the selections are not automatically calculated into some aspects. Such as programs increase deck speed will not update the deck itself. So keep this in mind.

Instructions 

Deck Manager
There are 4 options you can do with the manager:
(Clear) - This button will remove all the data in the cyberdeck and the program list. Be careful with this button ;-)
(Upload) - This button will allow you to select the saved deck manager json file to upload to this app.
(Save) - This button will save all deck manager data to a json file.
(PDF) - This button generates the deck manager to a PDF file listing the deck and the program list.

Cyberdeck Section
This section is self explanatory. To add/remove options click on to manage them.
MU is calculated as per source books. The check box is to add 10 more MU to stock. You will note that some printed deck has 15 or 25 MU. To acheive this, choose the Zetatech Memchip from options.
Price is calculated automatically. Some printed deck will be different. That is most likely due to addtional details or reduce price due to large surplus.

Program List Section
The number next to the header are the total MU of programs loaded (checked) compared to the deck MU storage. The eb value is the total cost of all the programs from the list.
The program list is sorted by class and then name. The Loaded check box is to mark if the program is loaded into the deck. STR and MU are listed as well.
Hovering over the class name will show you the description of the class. Hovering over the name of the program will display its description. Click on the name will open the program to be modified.
will delete the program form the list.
At the bottom of the list, the will pop up a blank program dialog to add more programs to the list.
Program Pop - This is where you would enter or modify a program. You must click on the button to save the changes to the program list.
