Cyberpunk 2020 Character Generator

This is a utility to create a Cyberpunk 2020 character. Here you will be able to create, save, load, and print character sheets. There is still a bit of work to be done, but this should help with starting out with the basics.

PRINTING - To print, use the browser to print it. Make sure to print the Background Graphics as well. This is usually off by default.

  • Chrome - add the background colors to print go Print > More settings > options > Background graphics.
  • Edge - doesn't support background color, but the form will print. IE does but the site doesn't work in IE.
  • Firefox - Page setup > Print Background. You will also need to set the scale of the page to 70%. There is an issue with Firfox distorting some of the elements while printing.

SAVE/UPLOAD - The data is save into a .txt file. This is the JSON object of the character. If you want to mess with it, back the file up before you edit it. This is a free app and created in my spare time so if you broke it you bought it ;-).

If you see icon, click it to get dice roll for a skill or save, or generate random lifepath.
