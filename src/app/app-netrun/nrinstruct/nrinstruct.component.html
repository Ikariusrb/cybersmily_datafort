Netrun Navigator Instructions

  • Grid Square
    Clicking on a grid square will shade/unshade it to mark the path of a run. You may click anywhere on the map for an initial location, then you must click on an adjacent square to track your movement.

  • LDLS
    To use this map hover over icon for the name of the city. Clicking on the city icon / will get more information about the LDL. If the current space is is selected (highlight in red) then option to HACK it will appear.

  • DATAFORTS
    These represent an actual datafort the runner will hack into. The DIFF is given for the Quick Netrun Rules from this site (go to Mods > Quick NR). The info will also indicate if an AI and/or Sysop is in the fort.


  • Clicking on "Total Trace" will show the LDLs that have been hacked. This includes the name, region, city, and the trace value. The total will also be displayed.


  • Click on "Hack" button to try and use the LDL's trace value. The current total trace appears in the Total Trace button above. When hacking, a roll for security will be done. Some events will clear your traces as it dumps you from the NET. If successful, the trace for that LDL will be added.
    NOTE: Security has change to have a 1d10 roll higher than the security number. So 1 is no longer an automatic success.

  • Region City World
    Click on one of these buttons will open the next map.


  • This button will reset all the maps and trace values.

  • Enhancement/Bugs/Updates
    This is still a work in progress. Some LDLs are misbehaving. Maps for cities and orbitville haven't been create yet. Stay tuned for those...

  • Resources
    For more information consult Cyberpunk 2020 mainbook and Rache Bartmoss' Guide to the Net available from DriveThru RPG. I strongly suggest picking up a copy of both, especially the guide as it has a lot of info about the regions.
