NET Architecture Generator for Cyberpunk Red will randomly generate an architect to be used in your games. Some of the options that are generated are very generic. This is to keep with R Talsorian's Homebrew policy.

To use this app, click the dice () button to generate a random Net architect. By default the difficulty and number of floors will be randomly generated. To specify a difficulty or number of floors, uncheck the box next to the selection and enter the desired value. A diagram of the architect will be shown on the left and the key to the diagram will be on the right. Scroll over the title of the node to show the details of it.

The icons represent the node types within the architect.

  • = Password
  • = File
  • = Control Node
  • = Program

To edit a node, click on the pen (). This opens the edit form to update and change the details of the node.

- NODE SETTINGS Button : This will only you to set the default colors for the type of nodes.

- SAVE Button : This will save a json file that can be uploaded later for editing.

- UPLOAD Button : This will upload the saved json file. If you try a different json file it will not load.

- SAVE AS IMAGE Button : This will save the diagram has a SVG image file. This will not be able to be uploaded.

Note on PDF: I am afraid I haven't figure out a way to embed the SVG image into a PDF using the utility libraries for this site. I am working on a way to do this. Until then you should be able to use Chrome or Edge to print as PDF. I have tested it on both browsers and seems to be a good work around for now. Sorry Firefox users :(
